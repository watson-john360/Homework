Landing Code using Phabricator

There are two main ways of landing code through Phabricator; using the command line arcanist API, and via the web-based GUI. When using arcanist command line scripts, there are several commands that behave similarly to standard git commands (modified for the Phabricator pipeline).

  • arc diff send your code for review, testing ling and unit first.
  • arc patch applies changes in a revision to the working copy.
  • arc land push changes to the server.
  • arc lint checks code for syntax and style errors.
  • arc unit runs unit tests on your changes.

The full listing of Arcanist API commands can be found in the arcanist users' guide.
